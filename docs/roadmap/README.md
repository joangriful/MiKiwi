# <img src="../assets/icons/roadmap.svg" width="32" height="32" /> Roadmap del Proyecto

Esta sección detalla la trayectoria estratégica de MiKiwi, desglosando los hitos alcanzados y los objetivos futuros para asegurar el crecimiento sostenido del ecosistema.

### Resumen del Área
El roadmap sirve como brújula del proyecto, permitiendo a los desarrolladores y stakeholders visualizar el progreso técnico y funcional. Aquí se definen las prioridades, desde el core del e-commerce hasta la integración de funcionalidades 3D avanzadas y sistemas de fidelización.

# <img src="../assets/icons/epics.svg" width="32" height="32" /> Historias de Usuario y Épicas

Esta sección desglosa las funcionalidades principales de MiKiwi en forma de épicas e historias de usuario, definiendo el alcance funcional desde la perspectiva del cliente.

### Resumen del Área
Las épicas nos permiten organizar el desarrollo en grandes bloques de valor (ej: Configuración 3D, Checkout, Gestión de Perfil). Cada una se subdivide en tareas atómicas que guían la implementación y aseguran que cada funcionalidad cumpla con las expectativas de negocio y las necesidades del usuario final.

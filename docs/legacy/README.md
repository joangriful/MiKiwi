# <img src="../assets/icons/backup.svg" width="32" height="32" /> Documentación Antigua (Legacy)

Este repositorio de archivos contiene la documentación y referencias de versiones anteriores, sistemas reemplazados y auditorías históricas de MiKiwi.

### Resumen del Área
Mantener un registro de lo "legacy" es vital para entender la evolución del proyecto y evitar repetir errores pasados. Aquí se conservan los esquemas de base de datos antiguos, planes de migración ejecutados y cualquier documento que, aunque ya no sea la "fuente de verdad" actual, proporcione contexto histórico valioso para el equipo.

---

## 📂 Recursos Históricos

*   **[Bases de Datos Legacy](database/)**: Esquemas y modelos relacionales de las fases iniciales del proyecto.
*   **[Documentación Archivada](.)**: Referencias técnicas de sistemas y módulos ya migrados o dados de baja.

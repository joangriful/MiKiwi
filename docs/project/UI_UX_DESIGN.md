# Guías de Diseño y UI

MiKiwi prioriza una interfaz elegante, discreta y altamente funcional, centrada en el protagonismo del producto.

## Principios de Diseño
*   **Elegancia y Claridad**: Uso de espacios en blanco y tipografía legible.
*   **Contraste Óptimo**: Garantizar que todos los elementos de control sean visibles y accesibles.
*   **Responsive Design**: Adaptación fluida a dispositivos móviles, tablets y escritorio.
*   **Estados Interactivos**: Definición clara de estados `hover`, `focus`, `active`, `disabled` y `loading`.

## Convenciones Técnicas
*   **CSS Modules**: Técnica principal para estilos locales de componentes.
*   **Global CSS**: `resources/css/global.css` reservado exclusivamente para tokens de diseño (colores, fuentes) y resets.
*   **Tailwind CSS**: Uso limitado a utilidades puntuales o componentes legacy en proceso de migración.
*   **Accesibilidad**: Foco en navegación por teclado y cumplimiento de estándares básicos de contraste.

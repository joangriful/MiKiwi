# Stack Tecnológico MiKiwi

El proyecto utiliza un stack moderno y profesional, equilibrando rendimiento en el cliente con robustez en el servidor.

## Backend (Laravel Ecosystem)
*   **Laravel 12**: Core del framework.
*   **PHP 8.2+**: Tipado estricto y alto rendimiento.
*   **Inertia Laravel**: Puente para renderizado de componentes React sin API REST.
*   **Laravel Sanctum**: Autenticación y seguridad de tokens.
*   **Ziggy**: Integración de rutas Laravel en el frontend.
*   **Stripe PHP SDK**: Gestión de pagos.
*   **Cloudinary PHP**: Gestión remota de assets.

## Frontend (Modern React)
*   **React 18**: Librería de interfaz de usuario.
*   **Inertia React**: Integración con el backend.
*   **Vite**: Herramienta de build y HMR ultra-rápido.
*   **CSS Modules**: Estilado local y modular (convención principal).
*   **React Toastify**: Notificaciones de sistema no bloqueantes.
*   **Framer Motion / Lenis**: Animaciones fluidas e scroll suave.

## 3D y Multimedia
*   **Three.js**: Motor 3D base.
*   **React Three Fiber / Drei**: Abstracción de Three.js para React.
*   **Cloudinary**: CDN y gestión de assets multimedia.
*   **React Easy Crop**: Procesamiento de imágenes en cliente.

## Persistencia
*   **PostgreSQL**: Base de datos relacional principal.
*   **Supabase**: Entorno remoto gestionado para producción y dev.
*   **Local PostgreSQL**: Entorno aislado para ejecución de tests automatizados.

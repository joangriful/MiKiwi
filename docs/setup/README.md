# <img src="../assets/icons/instalacion.svg" width="32" height="32" /> Configuración e Instalación

Esta sección proporciona las instrucciones necesarias para poner en marcha el entorno de desarrollo de MiKiwi y configurar los servicios externos críticos para su funcionamiento.

### Resumen del Área
Para garantizar una experiencia de desarrollo fluida y consistente, es fundamental seguir los pasos de instalación y configuración detallados en estos documentos. Cubrimos desde la preparación del entorno local (PHP, Node, PostgreSQL) hasta la integración de servicios en la nube como Cloudinary para la gestión de activos multimedia.
